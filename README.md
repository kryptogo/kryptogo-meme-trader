# KryptoGO Meme Trader

AI-powered meme coin trading agent for Solana with multi-chain on-chain analysis.

## Packages

### [Skill](./skill/)

Claude Code skill for meme coin analysis and trading. Published to [ClawHub](https://clawhub.com).

```bash
# Install via ClawHub
npx clawhub install kryptogo-meme-trader
```

See [skill/SKILL.md](./skill/SKILL.md) for full documentation.

### [MCP Server](./mcp-server/) (coming soon)

KryptoGO MCP server for token analysis and trading. Will be published to npm.

## Publishing

Skills are published automatically when a version tag is pushed:

```bash
git tag v2.5.1
git push origin v2.5.1
```
